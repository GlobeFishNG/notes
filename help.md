# Help

## UML

Using [PlantUML](https://github.com/GitbookIO/plugin-puml) to draw UML.

* SVG output. \(Friendly to scaling\)

```text
{% plantuml format = 'svg' %}
Alice -> Bob: Hello, Bob
Alice <-- Bob: Hello, Alice
{% endplantuml %}
```

* PNG output.

```text
{% plantuml format = 'png' %}
Alice -> Bob: Hello, Bob
Alice <-- Bob: Hello, Alice
{% endplantuml %}
```
